Welcome to [Hexo](https://hexo.io/)! 这是一篇典型的默认示例文章，也是旧博客历史里最容易确认的一篇内容。把它保留下来，不只是为了纪念当时的起点，也是在提醒自己：真正可持续的站点，应该把“内容”和“发布结果”分开管理。

## Quick Start

### Create a new post

```bash
hexo new "My New Post"
```

更多内容可以参考官方文档中的 [Writing](https://hexo.io/docs/writing.html)。

### Run server

```bash
hexo server
```

更多内容可以参考 [Server](https://hexo.io/docs/server.html)。

### Generate static files

```bash
hexo generate
```

更多内容可以参考 [Generating](https://hexo.io/docs/generating.html)。

### Deploy to remote sites

```bash
hexo deploy
```

更多内容可以参考 [Deployment](https://hexo.io/docs/one-command-deployment.html)。
