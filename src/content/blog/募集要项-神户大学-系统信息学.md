### 神户大学系统信息学募集要项解读
原始募集要项：
https://www.csi.kobe-u.ac.jp/exam/master/2026/x_master_ippan_202608.pdf

这份 PDF 同时包含：

- `一般入试`
- `外国人留学生特别入试`

这篇文章只解读你这次真正关心的 `一般入试`。有些要求在特别入试里出现，但在一般入试里并没有出现，不能混着看。

另外，募集要项明确写了：文中所有时间都按 `日本时间` 计算。虽然这是 `2027 年 4 月入学` 的考试，但真正的出愿、考试和放榜都发生在 `2026 年`。

### 先看最重要的结论

- 神户大学大学院系统信息学研究科博士课前期课程，本次只有一个专攻：`系统信息学专攻`
- 募集人員写的是 `103 人`，但这个数字 `包含推荐入试和外国人留学生特别入试`
- 一般入试的 `Web 出愿登记 + 检定料支付` 时间是 `2026-07-15 00:00` 到 `2026-07-24 16:59`
- 需要书面提交的材料，邮寄截止是 `2026-07-24 17:00 必着`，不是消印有效
- 走出愿资格 `(9)~(11)` 的人，要先做资格审查，时间是 `2026-06-09` 到 `2026-06-11`
- 英语不是校内考，而是直接采用 `TOEFL iBT` 或 `日本国内 TOEIC L&R 公开测试`
- 有效英语成绩必须是 `2024-08-24` 以后参加的考试
- 一般入试要参加 `数学 + 专业课 + 口头试问`
- 笔试在 `2026-08-24`，口试在 `2026-08-25`
- 合格发表时间是 `2026-09-04 10:00`

如果只记一件事，那就是：这份募集要项最容易坑人的地方不是考试本身，而是 `英语成绩提交方式` 和 `书面材料必须在截止前送达`。

### 这份一般入试到底在招什么

募集要项第一页写得很直白：

- 研究科：`神户大学大学院系统信息学研究科`
- 课程：`博士课前期课程`
- 方式：`一般入试`
- 专攻：`系统信息学专攻`
- 募集人数：`103 人`

但这里的 `103 人` 不是“一般入试单独拿到 103 个名额”的意思，因为原文同时注明，这里面已经把：

- `推荐入试`
- `外国人留学生特别入试`

一起算进去了。

所以如果你在估算竞争时直接把 `103` 当作一般入试纯名额，通常会偏乐观。

### 最关键的时间线

| 时间 | 事项 | 备注 |
| --- | --- | --- |
| `2026-06-09` 至 `2026-06-11` | 出愿资格审查申请 | 仅适用于出愿资格 `(9)~(11)` |
| `2026-06-11 17:00` | 资格审查材料到达截止 | 邮寄也必须在这个时间前到达 |
| `2026-06-30` 前 | 资格审查结果通知 | 学校通知本人 |
| `2026-07-15 00:00` 至 `2026-07-24 16:59` | Web 出愿登记 + 检定料支付 | 先在 `e-apply` 完成 |
| `2026-07-15` 至 `2026-07-24 17:00` | 书面材料邮寄提交 | `书留速达`，而且是 `必着` |
| `2026-07-21` 至 `2026-07-24` | 现场提交窗口开放 | `9:30-16:00`，中午和休息日除外 |
| `2026-08-17` 前后 | 受验票邮件通知 | 如果到这天还没收到邮件，要联系教务 |
| `2026-08-24 09:30-10:50` | 数学笔试 | 不是集合时间 |
| `2026-08-24 12:30-13:50` | 专业课笔试 | 不是集合时间 |

| `2026-09-04 10:00` | 合格发表 | 在 Web 出愿网站的 My Page 查看 |
| `2027-02` 下旬 | 入学手续通知寄出 | 学校邮寄 `入学の手引き` |
| `2027-03` 中旬 | 入学手续期间 | 详细日程以后续通知为准 |

神户大学这条线和很多学校最大的不同之一，就是它把 `到达截止` 写得非常明确。你不能按“最后一天寄出去就行”来规划。

### 你是不是需要先做出愿资格审查

不是所有人都需要。

如果你是比较常见的路径，比如：

- 普通四年制本科毕业或预计毕业
- 在国外完成 16 年学校教育
- 通过国外三年以上课程取得相当于学士学位

一般就是按正常资格出愿。

真正要重点核对的是 `(9)~(11)`：

- `(9)` 通过学校教育法第 102 条第 2 项路径提前进入大学院的人
- `(10)` 通过个别资格审查认定具有大学毕业同等以上学力的人
- `(11)` 在入学时满足“大学在学 3 年以上”或“15 年教育经历”等条件，并且被认定修得成绩优异的人

这三类人都要先做资格审查，时间统一是：

- `2026-06-09` 到 `2026-06-11`
- 邮寄的话 `2026-06-11 17:00 必着`
- 结果 `2026-06-30` 前通知

这里还有一个很容易忽略的点：

`出愿资格 (11)` 的人，后面不是直接正常录取，而是先做 `仮合格` 判定，之后还会在 `2027 年 3 月` 进行最终审查。也就是说，这条路径的“9 月放榜”并不等于一切已经完全结束。

### 正式出愿怎么走

神户大学把一般入试的出愿拆成两步：

#### 1. 先做 Web 出愿登记和检定料支付

时间是：

- `2026-07-15 00:00` 到 `2026-07-24 16:59`

入口是：

- `https://e-apply.jp/ds/kobe-u/`

这一步不是简单填个表，而是要按研究科官网给的 `Web 出愿の流れ` 和 `Web 出愿サイト入力の手引き` 去做。

#### 2. 再提交需要书面的材料

邮寄提交：

- `2026-07-15` 到 `2026-07-24`
- 截止标准是 `2026-07-24 17:00 必着`
- 邮寄方式必须是 `书留速达`

现场提交：

- `2026-07-21` 到 `2026-07-24`
- `9:30-16:00`
- 土日祝和 `12:00-13:00` 不受理

这里最重要的不是“寄了没有”，而是“按时到没到”。如果你从海外寄材料，更不能卡最后几天。

### 一般入试到底要交哪些材料

这份募集要项不是所有材料都纸质，也不是所有材料都上传。它是 `Web 上传 + 书面提交` 混合模式。

#### Web 端要处理的内容

- `脸照数据`
- `研究计划书`
- `出愿登记及出愿书类确认表`
- `住民票`（仅限在日本居留的外国籍志愿者）

其中研究计划书有几个关键词特别重要：

- 必须用研究科指定模板
- 从官网页面下载模板
- `1 页以内`

#### 书面提交的核心材料

- `TOEFL` 成绩单，或者 `TOEIC L&R 公开测试` 成绩
- `成绩证明书`
- `毕业（预计毕业）证明书` 或 `修了（预计修了）证明书`
- `受验许可书`（仅限在读大学院生或在职者）
- `国费外国人留学生证明书`（仅限适用者）
- `学位授与证明书` 或 `学位授与申请见込证明书`（仅限按资格 `(2)` 出愿者）

募集要项还明确写了几个底线规则：

- 原则上提交的书类都要 `原件`
- 没有特别写明可复印的，`复制件不认可`
- 书类不全或填写有误，学校 `不受理`
- 非日语或英语材料，要附 `公的机构出具的日语译文或英语译文`

### 如果你是中国学校出身，这里有一个额外坑

这份募集要项专门单列了中国大学证明书的要求。

如果你提交的是 `中国大学的毕业（修了）证明书`，除了学校原件外，还要通过 `CHSI` 做英文认证，并且让电子认证邮件 `直接发送到神户大学系统信息学研究科`。

它还明确写了两件事：

- 认证发送可能很慢，假期会更慢，要尽早申请
- 如果到入学时认证书还没到，`可能取消合格`

这部分不是“建议”，而是写进募集要项的正式要求。

### 英语要求怎么理解

这一条是一般入试的核心。

#### 1. 神户大学这里没有校内英语笔试

英语部分直接用外部成绩计分：

- `TOEFL` 或 `TOEIC L&R`

在配点表里，英语是：

- `100 分`

也就是说，英语不是参考项，而是正式计分项。

#### 2. 有效期不是“随便最近两年”，而是卡到具体日期

募集要项写的是：

- `2024-08-24` 以后参加的成绩才有效

所以不要只模糊记成“最近两年内”，最好直接按这个绝对日期判断。

#### 3. TOEFL 只收 iBT

可以提交的是：

- `TOEFL iBT`

不接受的是：

- `TOEFL ITP`
- `TOEFL iBT Home Edition`

提交方式也不只是把分数写上去，而是要：

- 交原本和 A4 复印件，或者
- 打印官网下载的 `Test Taker (Examinee) Score Report` PDF

而且后续还必须办理：

- 让 `Institutional Score Report` 发到神户大学
- DI Code 是 `B071`

#### 4. TOEIC 只收日本国内的公开测试

神户大学接受的是：

- `日本实施的 TOEIC Listening & Reading 公开测试`

明确不接受的是：

- 日本国外举行的 TOEIC
- `TOEIC-IP`
- 团体受验类成绩

而且这里不是上传截图，也不是自己打印一张成绩单就行。学校要求你走 `TOEIC「公开测试スコア确认服务」`，从 TOEIC 申込网站把成绩提交给神户大学。

神户大学在募集要项里给了专用申请代码：

- `00010408`

这一条非常关键，因为原文明确写了：

- 如果在出愿期间内，不能从 TOEIC 申込网站完成成绩提交
- 学校会视为 `スコア未提出`
- 进而认定 `出愿书类不备`
- 最终 `不认可出愿`

也就是说，TOEIC 这里不是“晚点补交”，而是 `没按规定提交通道完成，就可能直接失去报名资格`。

#### 5. 这份一般入试没有写统一最低分

募集要项没有给出统一的 TOEFL 或 TOEIC 最低分线。

但这不代表英语不重要。因为它本来就是正式计分项，而且占 `100 分`。

### 研究室志望怎么填

一般入试这部分写得也很明确：

- 志望讲座（研究室）在 Web 出愿网站填写
- 对照 `P.30-P.32` 的讲座（研究室）列表
- `必须填到第 6 志望`

这里有一个很实用的判断点：

一般入试正文只要求你在系统里填研究室志愿顺位，并 `没有` 写“必须先拿第一志望教员的受入内诺书”。这个要求出现在同一份 PDF 的 `外国人留学生特别入试` 部分，不应直接套用到一般入试上。

所以如果你是一般入试，不要把“受入内诺书”误当成必交件；但如果你自己想提前联系老师，那是策略层面的事，不是这份一般入试募集要项明文写出的硬性门槛。

### 考试怎么考

一般入试是：

- `数学`
- `专业科目`
- `英语`
- `口头试问`

其中配点是：

| 项目 | 配点 |
| --- | --- |
| 数学 | `150` |
| 专业科目 | `150` |
| 英语 | `100` |
| 合计 | `400` |

口头试问不计入这 `400` 分里，而是：

- `合 / 否` 判定

#### 1. 数学范围

募集要项给出的范围是：

- 线性代数
- 微积分
- 常微分方程
- 复素函数论

#### 2. 专业课范围

专业课是四个方向中选两个作答，每个方向 `75 分`：

1. `控制工学`
2. `数理计划`
3. `计算机系统`
4. `算法与数据结构`

这意味着它不是“全考四门”，而是从四个方向里选两个答题。

#### 3. 考试日程

| 日期 | 时间 | 科目 |
| --- | --- | --- |
| `2026-08-24` | `9:30-10:50` | 数学 |
| `2026-08-24` | `12:30-13:50` | 专业科目 |
| `2026-08-25` | `14:00-` | 口头试问 |

试验场是：

- `神户大学大学院工学研究科学舎`
- 地址：`神户市灘区六甲台町 1-1`

募集要项还写了一个容易被忽略的细节：

- `上述时间不是集合时间`

真正的集合时间、交通说明和考场注意事项，要看后面邮件发送的 `受验者心得`。

#### 4. 考试当天能带什么

配点表里写的是：

- `笔记用具以外的携带品：不可`

再配合注意事项里的表述，基本可以理解为不要默认自己能带计算器、电子表、智能设备等。手表也只允许 `纯计时功能` 的款式。

### 放榜、学费和入学手续

#### 1. 放榜

- `2026-09-04 10:00`
- 在 Web 出愿网站 `My Page` 查询

学校还特别强调：

- 不接受电话查询

#### 2. 检定料和入学后的费用

出愿时检定料：

- `30,000 日元`

入学后费用按募集要项当前记载是：

- 入学料：`282,000 日元`
- 学费前期：`267,900 日元`
- 学费年额：`535,800 日元`

不过募集要项也注明，这里写的是 `2026 年度的例`，后续有可能调整。

#### 3. 入学手续

- `2027 年 3 月中旬` 办理
- 详细内容在 `2027 年 2 月下旬` 通过邮寄的 `入学の手引き` 通知

### 这份募集要项里最容易踩坑的地方

如果把整份一般入试 PDF 压缩成几个最关键的风险点，我会认为是下面这些：

#### 1. `17:00 必着` 不是消印有效

不论是资格审查，还是正式出愿，只要原文写的是 `必着`，你就要按“提前送到”去规划，而不是按“截止当天寄出”去规划。

#### 2. TOEIC 不是传截图

神户大学要求你通过 TOEIC 官方的 `公开测试スコア确认服务` 提交。做不完这个动作，学校可能直接判定你 `书类不备`。

#### 3. 一般入试和外国人特别入试不能混着看

同一份 PDF 里，特别入试要求：

- 第一志望研究分野教员的 `受入内诺书`

但一般入试这部分没有这样写。你如果把两条线混在一起看，很容易多做一堆并非硬性要求的材料，或者反过来漏掉自己真正该交的东西。

#### 4. 103 人不是一般入试净名额

这个数字已经把推荐和外国人特别入试算进去了，不能直接拿来估算一般入试的独占名额。

#### 5. 中国学历材料有 CHSI 追加要求

如果你是中国高校出身，这一步要尽早做。它不是“补一封邮件就行”的小事，而是可能影响最终合格维持与否的正式要求。

#### 6. `(11)` 资格路径的人先拿到的是仮合格

如果你走的是 15 年教育或大学 3 年以上那条线，要特别注意它后面还有 `2027 年 3 月` 的最终审查，不是 9 月放榜就完全结束。

### 最后怎么用这份募集要项

如果你现在就在准备神户大学系统信息学的一般入试，我建议把事情拆成这几个顺序：

1. 先确认自己是不是普通资格，还是 `(9)~(11)` 需要先审查
2. 尽快确认英语到底交 `TOEFL` 还是 `TOEIC`
3. 如果你走 `TOEIC`，提前熟悉官方成绩提交流程，不要拖到出愿周
4. 提前准备研究计划书模板和各类证明书原件
5. 如果涉及中国学历材料，尽早处理 `CHSI`
6. 再回头集中准备数学、专业课和口试

这份募集要项本身并不算绕，但它对 `时间节点` 和 `提交方式` 的要求很硬。只要前面的手续不失误，后面你就可以把更多精力留给真正的笔试和口试准备。
