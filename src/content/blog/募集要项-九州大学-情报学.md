### 九州大学情报学募集要项解读

原始募集要项：

https://www.isee.kyushu-u.ac.jp/wp/wp-content/uploads/2027mc_general_guidelines_20260420.pdf

这份 PDF 是九州大学大学院系统情报科学府 `2027年4月入学` 修士课程的募集要项，覆盖：

- `一般試験`
- `特別試験`
- `情報理工学専攻`
- `電気電子工学専攻`

这篇文章只聚焦你更关心的这条线：

- 专攻：`情報理工学専攻`
- 方式：`一般試験`
- 入学时间：`2027年4月`

先说最重要的一点：虽然这是 `2027年4月入学` 的募集要项，但正式申请和考试都发生在 `2026年`。如果你把“入学年份”和“申请年份”混在一起，最容易错过的不是考试，而是前面的 PSD 和出愿。

### 先看最重要的结论

- 海外本科毕业或预计毕业者，原则上要先做 `PSD 出願前評価`
- 出愿资格 `(9)~(13)` 的人，还要先做 `出願資格事前審査`
- `一般試験` 走的是 `学科試験`，也就是笔试路线
- `情報理工学専攻` 的英语不是校内考，而是直接采用外部英语成绩
- 英语不交不会“延期处理”，而是直接按 `0分`
- 募集要项明确要求：不问出身大学，`必须` 事先联系希望指导教员

如果你只想抓主线，那就是这六件事：

1. 先确认自己是否需要做 PSD
2. 再确认自己是否属于 `(9)~(13)`，需要资格预审
3. 提前联系志望教员
4. 在 `2026-07-01` 到 `2026-07-09` 完成 WEB 出愿
5. 按要求提交英语成绩
6. `2026-08-25` 去伊都校区参加笔试

### 这份募集要项到底在讲什么

这份文件不是单独为情报理工学专攻写的，而是系统情报科学府修士课程的总募集要项。  
它同时覆盖一般试验和特别试验，也同时覆盖情报理工学专攻和电气电子工学专攻。

所以你在读的时候，最重要的不是把整份 PDF 全背下来，而是先筛出和你这条申请线直接相关的几块：

- 第 3 节：出愿资格
- 第 5 节：PSD 出愿前评价
- 第 6 节：出愿资格事前审查
- 第 9 节：出愿手续
- 第 10 节：英语资格考试成绩
- 第 13 节：选拔方法
- 第 14 节：考试科目和配点

如果你考的是 `情報理工学専攻・一般試験`，真正需要盯住的其实就是这些部分。

### 最关键的时间线

| 项目 | 时间 | 备注 |
| --- | --- | --- |
| PSD 出愿前评价申请截止 | `2026-04-26 23:00` | 原则上适用于海外本科毕业或预计毕业者 |
| 出愿资格事前 WEB 审查 | `2026-06-15` 至 `2026-06-19` | 仅适用于资格 `(9)~(13)` |
| 出愿资格事前审查结果通知 | `2026-06-30` 前 | 学校通知 |
| 一般试验 WEB 出愿期间 | `2026-07-01 10:00` 至 `2026-07-09 17:00` | 需要在系统内完成 |
| 英语成绩补交截止 | `2026-07-31 17:00` | 仅限出愿期内尚未拿到成绩单者 |
| 一般试验笔试 | `2026-08-25` | 伊都校区 |
| 一般试验合格发表 | `2026-09-17 10:00` | 官网与邮寄通知 |
| 入学时间 | `2027-04-01` | 修士入学 |

这条时间线里最容易被低估的是前两个节点：

- `2026-04-26` 的 PSD
- `2026-06-15` 到 `2026-06-19` 的资格事前审查

因为这两个都发生在正式出愿之前，而且一旦错过，后面就不是“补一下材料”的问题，而是直接没有办法按这次时间线出愿。

### 你是不是必须先做 PSD

如果你是海外大学学士课程毕业者，或者预计毕业者，这一条非常重要。

募集要项第 5 节写明：

> 外国の大学（学士課程）を卒業（見込み）の者は，原則，2026年4月26日（日）23時までに PSD 出願前評価の申請を行い，事前に出願可の審査結果を得たうえで出願すること。

这句话的意思不是“建议海外本科申请者先做一下评估”，而是：

- 原则上必须先做 PSD
- 先拿到可以出愿的结果
- 再进入正式出愿

这部分还有两个容易漏掉的补充条件：

- 过去年度申请过的 PSD 结果，`不能沿用`
- `2027年度` 的申请需要重新做

募集要项里给出的例外对象是：

- 已经作为研究生在九州大学大学院系统情报科学府在籍的人
- 已被录取为 `MEXT` 国费外国人留学生的人

如果你正好是这两类之一，才可以不按通常逻辑重新做 PSD。

### 你是不是还要做“出愿资格事前审查”

这个和 PSD 不是一回事，而且不是所有人都要做。

募集要项第 6 节写明，只有按出愿资格 `(9)~(13)` 出愿的人，才需要在正式 WEB 出愿前接受资格事前审查。

时间是：

- `2026-06-15` 到 `2026-06-19`

结果通知时间是：

- `2026-06-30` 前

和大多数普通本科申请者最相关的几类资格通常是：

- `(1)` 大学毕业，或到 `2027-03-31` 前毕业
- `(3)` 在外国完成 `16年学校教育课程`，或到 `2027-03-31` 前完成
- `(6)` 在外国大学等完成 `3年以上课程` 并取得相当于学士学位，或预计到 `2027-03-31` 前取得

如果你属于正常的 16 年教育背景，很多时候是直接按 `(1)` 或 `(3)` 理解，不一定会走到 `(9)~(13)`。  
但如果你是跳级、15 年教育、专门学校、特殊学历路径，或者依靠个别资格认定出愿，那就一定要认真核对这一节。

### 必须提前联系希望指导教员

这一条我会单独拎出来，因为它非常容易被当成“经验贴建议”，但其实是募集要项正文里的明确要求。

文件写的是：

> 出願を希望する者は，出身大学を問わず，必ず希望指導教員に事前にメール等で連絡を取ること。

这里的关键词是：

- `出身大学を問わず`
- `必ず`

也就是说：

- 不管你是不是九大本校生
- 不管你是不是海外申请者
- 只要你要出愿，就必须事先联系志望指导教员

这不是“套磁可做可不做”，而是官方写进募集要项里的前置动作。  
如果你还没开始联系老师，这件事的优先级其实比刷题更高。

### 一般试验到底怎么考

`情報理工学専攻` 的一般试验不是面试先行，而是 `学科試験`。

募集要项第 13 节把选拔方式分得很清楚：

- `一般試験`：通过学科试验实施
- `特別試験`：另有口述试验路线

所以如果你走的是一般试验，主线准备应该是：

- 英语成绩
- 数学笔试
- 专业课笔试

而不是把重心放在特别试验的研究计划或口试逻辑上。

### 英语成绩怎么要求

九州大学这份募集要项里，英语要求是 `情報理工学専攻` 一般试验最关键的一块之一。

#### 1. 英语直接计分，不交就是 0 分

第 10 节和第 14 节结合起来看，英语不是一个“有就更好”的参考项，而是正式配点的一部分。

- 英语配点：`100分`
- 使用方式：直接采用英语资格考试成绩

募集要项还明确写了：

- 如果没有在规定期限内提交英语成绩证明，则英语成绩按 `0分` 处理

这意味着“没来得及交”不会自动变成“缺考但不计入”，而是直接拉低总分。

#### 2. 可接受的考试类型

可接受的考试是：

- `TOEIC Listening & Reading Test`
- `TOEFL iBT`
- `IELTS Academic Module`

成绩必须满足的有效期是：

- 考试日期在一般试验出愿截止日 `2026-07-09` 往前推两年以内

#### 3. 各类成绩的具体要求

**TOEIC**

- 只接受 `日本国内` 举办的 `TOEIC Listening & Reading Test`
- 只接受 `DIGITAL OFFICIAL SCORE CERTIFICATE`

这也意味着：

- 海外考的 TOEIC L&R 不在可接受范围内
- 旧式纸质成绩证明扫描上传，不符合这里的要求

**TOEFL iBT**

- 接受 `TOEFL iBT Home Edition`
- 不接受 `MyBest Score`
- 必须通过 ETS 把官方成绩送到九州大学
- DI Code 是 `D990`
- 同时还要把成绩报告 PDF 上传到 WEB 出愿系统

**IELTS**

- 只接受 `Academic Module`
- 需要上传官方成绩单 PDF

#### 4. 明确不接受的成绩

募集要项里明确排除了这些类型：

- 日本国外参加的 TOEIC L&R
- 邮寄版 TOEIC Official Score Certificate 的扫描 PDF
- `TOEIC IP`
- `College TOEIC`
- `TOEFL ITP`
- `TOEIC Speaking / Writing` 系列
- `IELTS General Training`

#### 5. 出愿期内还没拿到英语成绩怎么办

九大这里给了一个有限补交窗口。

如果你在出愿期间已经参加过英语考试，但成绩单还没出来，可以：

- 先在 WEB 出愿系统填写考试日期
- 之后在 `2026-07-31 17:00` 前补交正式成绩证明

但如果你最后还是没交上，处理方式不是“待补材料”，而是：

- 英语成绩记 `0分`

### 一般试验要交哪些材料

对情报理工学专攻一般试验来说，最核心的材料可以按下面理解。

#### 1. 学籍与学历材料

- 成绩证明书
- 毕业证明书或预计毕业证明书
- 学士学位证明书

其中，学位证明并不是所有人都额外交一份独立文件，是否需要补充，要看你具体适用的资格条款和学校要求的出具形式。

#### 2. 出愿系统与支付相关材料

- WEB 出愿系统内的申请信息
- 入学检定料支付
- 出愿完成状态确认

募集要项特别提醒，出愿不是“把资料填了就算完”，而是要确认系统状态：

- `申請状況` 为 `申請済`
- `Application` 为 `Completed`

如果状态没到这一步，学校就不会当作正式受理。

#### 3. 英语成绩材料

- TOEIC / TOEFL / IELTS 对应成绩证明

#### 4. 身份证明与照片

- 护照或在留相关证明材料
- 证件照数据

募集要项对照片的要求也很细：

- 最近 3 个月内拍摄
- 正面上半身
- 无帽
- 比例为 `4:3`
- 文件格式为 `png` 或 `jpg`

#### 5. 资格补充材料

如果你属于需要资格事前审查的人，还要额外准备与资格认定相关的材料。  
这部分不适合一概而论，因为它取决于你走的是 `(9)~(13)` 中哪一种资格。

### 检定料和费用

#### 1. 出愿时的检定料

- `30,000日元`

募集要项里给出的支付方式包括：

- 信用卡
- 日本国内便利店
- 银行汇款

支付受理开始时间是：

- `2026-05-07`

也就是说，虽然正式 WEB 出愿是 7 月初，但检定料支付入口会更早开放。

#### 2. 入学后的费用

- 入学料：`282,000日元（予定）`
- 学费：`267,900日元 / 半年`
- 学费：`535,800日元 / 年`

这里的“予定”也意味着后续学校有可能调整金额，所以正式入学前最好再看一次最新通知。

### 信息理工学专攻的一般试验考什么

这部分是你后面真正备考时最直接会用到的内容。

#### 1. 英语

- 配点：`100分`
- 方式：直接使用英语资格考试成绩

#### 2. 数学

- 配点：`200分`
- 时间：`100分钟`

题目结构是四块里答三块：

- 线性代数
- 解析学 / 微积分
- 向量分析
- 概率 / 统计

其中前两块是固定核心：

- 线性代数
- 解析学 / 微积分

然后再从下面两块二选一：

- 向量分析
- 概率 / 统计

募集要项还特别说明：

- `解析学・微積分` 的范围包含微分积分、微分方程、复变函数
- 实际选哪一块，是考试当天再决定

这对备考策略很重要，因为它不是报名时锁死，而是你可以在当天按自己最有把握的组合答题。

#### 3. 专业课

- 配点：`200分`
- 时间：`120分钟`

从下面 6 个领域中选 2 个作答：

1. 电路
2. 信息论
3. 自动机与形式语言
4. 电磁气学
5. 算法
6. 计算机体系结构

这里和数学一样，也是考试当天再选。  
所以如果你现在还没完全定死专业课组合，也不必焦虑到“现在就只能押两门”，更合理的做法是先把你最稳的两门打实，再给第三门留一点机动空间。

### 考场与合格发表

一般试验的笔试时间是：

- `2026-08-25`

考场是：

- `九州大学伊都校区`

合格发表时间是：

- `2026-09-17 10:00`

募集要项同时说明：

- 官网会公布受验番号
- 正式结果仍以学校通知为准

### 这份募集要项里最容易踩坑的点

如果把整份 PDF 压缩成几个最关键的风险点，我会认为是下面这些。

#### 1. 海外本科申请者别忽略 PSD

这一步不是出愿前可选动作，而是原则上的前置要求。  
而且旧年度结果不能沿用。

#### 2. PSD 和资格事前审查不是同一件事

很多人第一次看会把这两件事混在一起：

- PSD：主要针对海外本科背景的出愿前评价
- 资格事前审查：主要针对资格 `(9)~(13)` 的个别资格确认

它们对象不同，时间也不同。

#### 3. 英语不交就按 0 分

这比“缺材料会不会被提醒”严重得多。  
因为它不是行政补件问题，而是直接进入计分处理。

#### 4. 九州大学明确要求你提前联系希望指导教员

这一条不是经验贴里的通用建议，而是募集要项正文里的要求。

#### 5. 一般试验和特别试验别混着准备

这份文件同时写了两种路线。  
如果你考的是 `一般試験`，核心任务是英语、数学、专业课笔试，不要把时间误投到特别试验那套逻辑里。

### 如果你现在要开始准备，最合理的顺序是什么

如果按这份募集要项倒推，我会建议把任务拆成下面四步。

#### 第一步：先判断自己属于哪种申请路径

你需要先确认：

- 自己是不是海外本科背景，需要 PSD
- 自己是不是属于 `(9)~(13)`，需要资格事前审查
- 自己报的到底是不是 `情報理工学専攻・一般試験`

这一步决定了你后面所有节点，而不是一个可有可无的“背景确认”。

#### 第二步：马上联系志望教员

因为这不是后期加分动作，而是学校明文要求。

#### 第三步：并行准备英语和基础材料

你至少要尽早推进：

- 英语考试
- 成绩证明书
- 毕业或预计毕业证明
- 护照/在留相关材料
- 证件照

这里最容易拖的是英语和学校证明件，而这两样偏偏都不能在最后一周临时解决。

#### 第四步：按笔试结构安排复习

对情报理工学专攻一般试验来说，最现实的复习主线是：

- 数学先抓 `线代 + 微积分`
- 再在 `向量分析 / 概率统计` 里选更稳的一条
- 专业课先选你最有把握的两门

### 我的整体判断

九州大学这一份 `2027年4月入学` 的修士课程募集要项，表面上看是“信息很全”，但真正难的地方在于它把多条申请路线写在同一份 PDF 里。  
如果你不先把自己锁定到 `情報理工学専攻・一般試験` 这一条线上，就很容易把 PSD、资格事前审查、特别试验、英语要求、笔试安排这些内容混成一团。

对大多数准备九大情报学一般入试的同学来说，最该优先落实的不是“再等等看别人怎么说”，而是这三件事：

1. 明确自己是否需要 PSD 或资格事前审查
2. 尽快联系志望指导教员
3. 把英语成绩和基础材料准备到能在 7 月初顺利提交的状态

只要这三步不卡住，后面的备考才真正有意义。
