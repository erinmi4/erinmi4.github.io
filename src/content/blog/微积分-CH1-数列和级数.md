#### 比值判定法

在数列与级数的学习中，判定 $\frac{a^n}{n!}$ 的极限是一个非常经典的课题。你选择 $\infty$ 是一个直觉上的常见错误，通常是因为忽略了**阶乘（Factorial）** 与 **幂函数（Power Function）** 在趋于无穷大时增长速度的本质差异。

> 阶乘与幂函数真的太适合比值判定法了。

以下是详细的解释：

###### 1. 增长速度的较量 (Growth Rates)

在数学中，不同类型的函数趋向无穷大的“快慢”是有严格排序的。对于任何非零常数 $a$：

- **幂函数 ($a^n$)**：其增长速度是几何级的。
- **阶乘函数 ($n!$)**：其增长速度远超幂函数。

当 $n$ 足够大时，$n!$ 的增长会完全“碾压” $a^n$。因为分母（$n!$）变大的速度比分子（$a^n$）快得多，所以整个分式最终会趋向于 $0$。

###### 2. 直观的数值演示 (Intuitive Demonstration)

假设常数 $a = 10$，我们来看看这个数列的项是如何变化的：

- 当 $n=5$ 时：$\frac{10^5}{5!} = \frac{100,000}{120} \approx 833$（此时看起来确实在变大）
- 当 $n=10$ 时：$\frac{10^{10}}{10!} \approx 2,755$
- 当 $n=20$ 时：$\frac{10^{20}}{20!} \approx 41,103$（依然在增长）
- **转折点出现**：一旦 $n$ 超过了 $a$（在这个例子中是 10），分母增加的倍数就开始超过分子。
- 当 $n=50$ 时：$\frac{10^{50}}{50!} \approx 10^{-15}$（已经变得极其微小，趋近于 0）

###### 3. 利用文档中的“比值判定法”证明

根据你提供的材料，我们可以利用**比值判定法 (d'Alembert's Test)** 的逻辑来严格证明：

设 $x_n = \frac{a^n}{n!}$，我们考察相邻两项的比值：

$$\left| \frac{x_{n+1}}{x_n} \right| = \left| \frac{a^{n+1}}{(n+1)!} \cdot \frac{n!}{a^n} \right| = \left| \frac{a}{n+1} \right|$$

现在取极限：

$$\lim_{n \to \infty} \left| \frac{a}{n+1} \right| = 0$$

由于比值的极限 $0 < 1$，根据级数收敛的判定准则，级数 $\sum \frac{a^n}{n!}$ 是**绝对收敛 (Absolute Convergence)** 的。根据级数收敛的**必要条件**：如果一个级数收敛，那么它的通项必须趋于 $0$。

因此：

$$\lim_{n \to \infty} \frac{a^n}{n!} = 0$$

###### 4. 总结你的误区

你可能认为 $a^n$ 作为一个指数爆炸增长的项会主导结果。但实际上：

- $a^n$ 每次只乘一个固定的常数 $a$。
- $n!$ 每次乘的是一个越来越大的数（$n+1$）。

无论 $a$ 有多大（哪怕是 $1,000,000$），只要 $n$ 增加到足够大（比如超过一百万零一），$n!$ 每一项增加的倍数就会永远超过 $a^n$ 增加的倍数。**这就是为什么极限不是 $\infty$ 而是 $0$。**

#### 有界单调数列收敛定理

> In Example 4, a sequence is defined by $a_1 = \sqrt{2}$ and $a_{n+1} = \sqrt{a_n + 2}$. What is the limit of this sequence? 

在文档的第 1.1 节中，**例题 4 (Example 4)** 展示了如何处理由递推公式定义的数列。该数列的极限确实是 **2**。

为了让你在复习这张闪卡时能够掌握背后的数学逻辑，以下是详细的解题步骤：

##### 1. 证明极限存在（判定准则）

在求解极限之前，必须先确定极限存在。文档利用了“**有界单调数列收敛定理**”：

- **证明有界性**：利用数学归纳法。
  - $a_1 = \sqrt{2} < 2$。
  - 假设 $a_k < 2$，则 $a_{k+1} = \sqrt{a_k + 2} < \sqrt{2 + 2} = 2$。
  - 因此，数列 $\{a_n\}$ 有上界 2。
- **证明单调性**：考察 $a_{n+1}^2 - a_n^2 = (a_n + 2) - a_n^2 = -(a_n - 2)(a_n + 1)$。
  - 因为 $a_n < 2$ 且 $a_n > 0$，所以该式结果始终 $> 0$，即 $a_{n+1} > a_n$。
  - 因此，数列 $\{a_n\}$ 单调增加。
- **结论**：单调增加且有上界的数列必收敛。

##### 2. 计算极限值

既然已知极限存在，我们可以设 $\lim_{n\to\infty} a_n = \alpha$。

对递推公式 $a_{n+1} = \sqrt{a_n + 2}$ 两边同时取极限：

$$\alpha = \sqrt{\alpha + 2}$$

两边平方得到：

$$\alpha^2 = \alpha + 2$$

$$\alpha^2 - \alpha - 2 = 0$$

$$(\alpha - 2)(\alpha + 1) = 0$$

解得 $\alpha = 2$ 或 $\alpha = -1$。由于数列各项均为正数且单调增加，极限必须大于 0，因此排除 $-1$。

**最终极限值为 2。**

------

**闪卡背面补充建议：**

- **核心逻辑**：先证存在（单调有界），再设极限为 $\alpha$ 解方程。
- **类似结构**：这种“嵌套根号”的形式 $\sqrt{c+\sqrt{c+\sqrt{c+\dots}}}$ 的极限通常是方程 $x^2 - x - c = 0$ 的正根。

### 夹逼定理

**夹逼定理**（挤压定理）：若 $a_n \le c_n \le b_n$ 且 $\lim_{n \to \infty} a_n = \lim_{n \to \infty} b_n = a$，则 $\lim_{n \to \infty} c_n = a$。
