# 第七章：微分方程 (Differential Equations)

微分方程本质上是用来求 y 和 x 的关系的一种方式。
也就是 $f(y)=f(x)$,但是中间拥有的内容只有导数 $y'=dy/dx$。
微分方程从大类上分为一阶微分方程和二阶微分方程,
一阶微分方程
1. 分离变量法，主要是将x与y各自放一边，特别注意，对于 (ax+by)经常出现的情况应该使用 $u = x + y$
2. 齐次方程，也就是次数相同的情况，这种需要  set u as  y/x.
3. 线性情况：$dy/dx + Q(x)y = P(x)$ 
对于这种情况需要先求 等于 0 的情况
然后 set C as C(X),返回去求出$C(X)$的解答就可以了
4. 非线性情况，也就是所谓的伯努利情况$$y' + P(x)y = Q(x)y^n$$
这种情况需要转化为线性情况，也就是通过使用两边除以y来得到答案并且记得
set $u = y^{1-n}$

微分方程的作用在于：当我们已知一个函数的导数关系（变化规律）时，通过建立方程来求出原函数 $y(x)$。

## 💡 引入实例：牛顿冷却定律
**问题描述**：室温 $25^\circ C$，在 $t=0$ 时，水温 $T=70^\circ C$。求水温 $T$ 随时间 $t$ 的变化规律 $T(t)$。 **建立方程**：

$$\frac{dT}{dt} = -k(T - 25)$$

*注：这里我们已知变化率（导数），目的是求出* $T(t)$ *的具体表达式。*



## 一、 一阶微分方程 (First-Order DE)

### 一阶微分方程分类与求解对照表

> 一个关于 y*y* 的方程是线性的，必须满足以下**三个“不许”**：
>
> - **不许相乘**：y*y* 和它的导数不能乘在一起（例如：y⋅y′*y*⋅*y*′ 是绝对不行的）。
> - **不许有幂次**：y*y* 或 y′*y*′ 的指数只能是通常的 1。不能有 y2*y*2、(y′)3(*y*′)3 这种形式。
> - **不许嵌套（怕被“关起来”）**：y*y* 或 y′*y*′ 不能套在别的函数里面。比如 sin⁡(y)sin(*y*)、y′*y*′、ey*e**y* 就是典型的非线性，因为它们被三角函数、根号、指数函数“关在里面”了。

| **方程名称**                      | **方程特征（标准形式）**         | **求解方法 / 变换公式**                                      |
| --------------------------------- | -------------------------------- | ------------------------------------------------------------ |
| **一阶线性齐次微分方程**          | $y' = p(x) \cdot y$              | $y = Ce^{\int p(x)dx}$                                       |
| **一阶线性非齐次微分方程**        | $y' + p(x)y = q(x)$              | $y = e^{-\int p(x)dx} \left[ \int e^{\int p(x)dx} \cdot q(x)dx + C \right]$ |
| **伯努利方程**                    | $y' + p(x)y = q(x)y^n$           | 置换变量：$u = y^{1-n}$ 导数关系：$y' = \frac{y^n}{1-n}u'$   |
| **变量代换一：$u = \frac{y}{x}$** | $\frac{dy}{dx} = f(\frac{y}{x})$ | 代换：$y = ux$ 微分：$dy = udx + xdu$                        |
| **变量代换二：$u = ax + by + c$** | $\frac{dy}{dx} = f(ax + by + c)$ | 代换：$u = ax + by + c$ 微分：$dy = \frac{du - adx}{b}$      |

### 1. 可分离变量法 (Separation of Variables)

**适用形式**：$y' = f(x)g(y)$ 

**SOP (标准解题步骤)**：

1. 将 $y'$ 写成 $\frac{dy}{dx}$。
2. 将含 $y$ 的项移到等号一边，含 $x$ 的项移到另一边。
3. 两边同时积分。

**例题**：$y' = 2xy$

- 解：$\frac{dy}{dx} = 2xy \implies \frac{1}{y}dy = 2xdx$
- 两边积分：$\int \frac{1}{y}dy = \int 2xdx \implies \ln|y| = x^2 + C$
- 结果：$y = Ce^{x^2}$

### 2. 齐次方程 (Homogeneous Equations)

#### 变量替换法1

**特征**：方程中每一项的变量次数（指数和）相同。 

**解法**：令 $u = \frac{y}{x}$，即 $y = ux \implies \frac{dy}{dx} = u + x\frac{du}{dx}$。

**例题**：$(x^2 + y^2)dx - xydy = 0$

- 观察：各项均为 2 次。
- 变形：$\frac{dy}{dx} = \frac{x^2 + y^2}{xy} = \frac{x}{y} + \frac{y}{x}$
- 代入 $u$：$u + x\frac{du}{dx} = \frac{1}{u} + u \implies x\frac{du}{dx} = \frac{1}{u}$
- 分离变量：$u du = \frac{1}{x} dx \implies \frac{1}{2}u^2 = \ln|x| + C$
- 还原：$\frac{y^2}{2x^2} = \ln|x| + C$

#### 变量替换法2

**特征**：方程中有 $x+y$存在	

变量替换法二：将$(𝑎x + 𝑏y + 𝑐)$替换为$u$

**例题：** 解微分方程 $dy = (3x + 2y + 1)^2 dx$

------

**1. 变量代换**

令 $u = 3x + 2y + 1$，则对两边求微分得：

$$du = 3dx + 2dy$$

由此可导出 $dy$ 的表达式并代入原方程：

$$dy = \frac{du - 3dx}{2}$$

**2. 建立新方程**

将上述代换关系代入原微分方程：

$$\frac{du - 3dx}{2} = u^2 dx$$

整理得可分离变量的形式：

$$\frac{du}{2u^2 + 3} = dx$$

**3. 积分求解**

对等式左右两边求不定积分：

$$\int \frac{1}{2u^2 + 3} du = \int dx$$

计算得出：

$$\frac{\sqrt{6}}{6} \arctan \left( \frac{\sqrt{6}}{3}u \right) = x + C$$

**4. 还原变量**

将 $u = 3x + 2y + 1$ 代回上式，得到原方程的通解：

$$\frac{\sqrt{6}}{6} \arctan \left[ \frac{\sqrt{6}}{3}(3x + 2y + 1) \right] = x + C$$
### 一种解决特殊分式的一阶微分方程方法
这张图片详细介绍了形如 $\frac{dy}{dx} = f\left( \frac{ax + by + c}{px + qy + r} \right)$ 的微分方程解法。这种方程的解题核心在于通过**坐标平移**或**变量代换**，将其转化为我们熟悉的“可分离变量形”或“同次形（齐次形）”。

以下是根据图片内容总结的解题技巧：

---

#### 核心判定准则：计算行列式值 $aq - bp$

解题的第一步是观察分子分母中 $x$ 和 $y$ 的系数比例。

##### 1. 当 $aq - bp = 0$ 时（系数成比例）

* **特征**：两条直线 $ax + by + c = 0$ 与 $px + qy + r = 0$ **平行**。此时分子分母中的变量部分可以写成倍数关系，即 $\frac{a}{p} = \frac{b}{q} = k$。
* **技巧**：使用**简单的变量代换**。
* 令 $u = ax + by$ 或 $u = px + qy$（通常选简单的那个）。
* 求出 $\frac{du}{dx}$，代入原方程。


* **结果**：方程会转化为**变量分离形**，直接积分即可。
* **例题(1)解析**：$(x+y+1) + (2x+2y-1)y' = 0$。这里 $x+y$ 与 $2x+2y$ 成比例，令 $u = x+y$ 即可化简。

##### 2. 当 $aq - bp \neq 0$ 时（系数不成比例）

* **特征**：两条直线 $ax + by + c = 0$ 与 $px + qy + r = 0$ **相交**。
* **技巧**：使用**坐标平移（结构转换）**。
* **第一步**：解方程组 $\begin{cases} ax + by + c = 0 \\ px + qy + r = 0 \end{cases}$，设解为 $(x, y) = (\alpha, \beta)$。这个点就是原点平移的目标。
* **第二步**：令 $x = u + \alpha, y = v + \beta$。由于 $\alpha, \beta$ 是常数，所以 $dx = du, dy = dv$。
* **第三步**：代入原方程，常数项 $c$ 和 $r$ 会被抵消。


* **结果**：方程变为 $\frac{dv}{du} = f\left( \frac{au + bv}{pu + qv} \right)$，这属于**同次形（齐次方程）**。接着令 $v = ku$ 即可按常规同次形方法求解。

---

#### 总结思维导图

> **看到分式线性构造 $\rightarrow$ 检查系数交叉相乘 $aq - bp$**
> * **等于 0** $\rightarrow$ 整体代换 $\rightarrow$ 变量分离。
> * **不等于 0** $\rightarrow$ 找交点 $(\alpha, \beta)$ $\rightarrow$ 平移坐标 $\rightarrow$ 变同次形。

### 3. 一阶线性微分方程 (Linear DE)

**标准形式**：$\frac{dy}{dx} + P(x)y = Q(x)$

- 若 $Q(x) = 0$：称为**齐次**线性方程（可用分离变量法）。
- 若 $Q(x) \neq 0$：称为**非齐次**线性方程。

**例题**：$(x^2 - 1)dy + (2xy - \cos x)dx = 0$

- **Step 1: 化为标准型**

  $$\frac{dy}{dx} + \frac{2x}{x^2-1}y = \frac{\cos x}{x^2-1}$$

- **Step 2: 解对应的齐次方程**（分离变量） $\frac{dy}{dx} + \frac{2x}{x^2-1}y = 0 \implies y_h = \frac{C}{x^2-1}$

- **Step 3: 常数变易法**（代回原式） 设 $y = \frac{C(x)}{x^2-1}$，求导代入得 $C'(x) = \cos x$ 则 $C(x) = \sin x + C$

- **最终结果**：$y = \frac{\sin x + C}{x^2-1}$

### 4. 克罗莱方程情况
这张图片展示的是微积分中一类非常特殊的微分方程——**克莱罗方程（Clairaut's Equation）**。

这种方程的奇妙之处在于，它不仅有一组整齐的“直线解”，还隐藏着一个由这些直线“围”出来的**曲线解**（即奇异解）。

> 本质上是一阶导数的幂次情况

---

#### 1. 什么是克莱罗方程？

克莱罗方程的标准形式是：


$$y = x\frac{dy}{dx} + f\left(\frac{dy}{dx}\right)$$


在图中给出的例子是：


$$y = x\frac{dy}{dx} + \left(\frac{dy}{dx}\right)^2 - 1$$


这里 $f(p) = p^2 - 1$（令 $p = \frac{dy}{dx}$）。

---

#### 2. 解题步骤拆解

##### 第一步：令 $p = \frac{dy}{dx}$ 并求导

将方程写成 $y = px + p^2 - 1$。为了求出 $p$ 的值，我们对等式两边同时对 $x$ 求导：


$$\frac{dy}{dx} = p'x + p + 2pp'$$


因为左边 $\frac{dy}{dx}$ 本身就是 $p$，代入后：


$$p = p'x + p + 2pp' \implies p'(x + 2p) = 0$$

这里出现了一个乘积等于 0 的情况，于是产生了两个分叉路径：

##### 第二步：寻找一般解（General Solution）

由 $p' = 0$ 得出：

* $p = C$ （常数）。
* 将 $p = C$ 直接代回原方程，得到：

$$\mathbf{y = Cx + C^2 - 1}$$


* **物理意义**：这是一组**直线族**。每一个不同的 $C$ 值都代表坐标系里的一条直线。

##### 第三步：寻找奇异解（Singular Solution）

由 $x + 2p = 0$ 得出：

* $p = -\frac{x}{2}$。
* 将这个 $p$ 代回原方程 $y = px + p^2 - 1$：

$$y = \left(-\frac{x}{2}\right)x + \left(-\frac{x}{2}\right)^2 - 1$$


$$y = -\frac{1}{2}x^2 + \frac{1}{4}x^2 - 1 = -\frac{1}{4}x^2 - 1$$


* **物理意义**：这是一个**抛物线**。

---

#### 3. 核心概念：包络线（Envelope）

看图片右侧的图 **6.4**：

* 那些密密麻麻的**直线**就是**一般解**。
* 那条蓝色的**抛物线**就是**奇异解**。

![image-20260508184905090](./%E5%BE%AE%E7%A7%AF%E5%88%86-CH7-%E5%BE%AE%E5%88%86%E6%96%B9%E7%A8%8B.assets/image-20260508184905090.png)

**重点：** 奇异解产生的曲线恰好与每一条直线解都相切。在几何上，这条曲线被称为直线族的**包络线**。

---

#### 4. 补充技巧：注意 6.5

图片底部的“注意”提供了一个更简便的方法：
如果你已经求出了一般解 $y = Cx + C^2 - 1$，想要求包络线，只需要对参数 $C$ 求偏导：

1. 对 $C$ 求导：$0 = x + 2C \implies C = -x/2$
2. 代回原式，直接就能得到奇异解 $y = -\frac{1}{4}x^2 - 1$。

这种方程在物理学（如光学中的焦散线）中非常有用。你之前研究的信号处理或嵌入式系统里，如果涉及到非线性的路径规划或包络检测，其实底层逻辑是相通的。

### 5. 一阶非线性微分方程（伯努利情况）

这是一道非常经典题目！观察一下，方程的形式是：

$$y' + P(x)y = Q(x)y^n$$

由于右边多了一个 $y^3$，这不再是线性方程，而是**伯努利方程（Bernoulli Equation）**。

虽然它不是一阶线性方程，但我们可以通过**变量替换**，把它转化为可以用“常数变易法”解决的形式。

#### 第一步：化“非线性”为“线性”

为了干掉右边的 $y^3$，我们将方程两边同时除以 $y^3$：

$$y^{-3}y' + \frac{1}{x}y^{-2} = x^2$$

**关键替换：** 设一个新的变量 $z = y^{1-3} = y^{-2}$。

对 $z$ 求导（注意链式法则）：

$$\frac{dz}{dx} = -2y^{-3}y' \implies y^{-3}y' = -\frac{1}{2}z'$$

代入原式，方程变成了关于 $z$ 的一阶线性方程：

$$-\frac{1}{2}z' + \frac{1}{x}z = x^2$$

整理成标准型：

$$\mathbf{z' - \frac{2}{x}z = -2x^2}$$

#### 第二步：解齐次方程（寻找变易的基础）

令右边为 0：


$$z' - \frac{2}{x}z = 0 \implies \frac{dz}{z} = \frac{2}{x}dx$$


积分得 $\ln |z| = 2\ln |x| + \ln |C|$，即：


$$z_h = Cx^2$$

---

#### 第三步：常数变易法

现在，我们把常数 $C$ 变易为函数 $C(x)$，设：


$$z = C(x)x^2$$


求导：$z' = C'(x)x^2 + 2xC(x)$。代入刚才的线性方程 $z' - \frac{2}{x}z = -2x^2$：

$$(C'(x)x^2 + 2xC(x)) - \frac{2}{x}(C(x)x^2) = -2x^2$$

$$C'(x)x^2 + \color{red}{2xC(x)} - \color{red}{2xC(x)} = -2x^2$$

$$C'(x)x^2 = -2x^2 \implies C'(x) = -2$$

积分得：


$$C(x) = -2x + C$$

所以，$z$ 的通解为：


$$z = (-2x + C)x^2 = Cx^2 - 2x^3$$

---

#### 第四步：还原变量

别忘了我们最初设的是 $z = y^{-2}$，所以：


$$\frac{1}{y^2} = Cx^2 - 2x^3$$


或者写成：


$$y^2 = \frac{1}{Cx^2 - 2x^3}$$

---

#### 总结

处理这种 $y' + Py = Qy^n$ 的情况，思路是：

1. **除以 $y^n$**。
2. **令 $z = y^{1-n}$** 降维打击，把它变回一阶线性方程。
3. **常数变易法** 此时就可以像之前一样大显身手了。

这种通过变换把“没见过的问题”变成“已解决的问题”，也是你研究嵌入式算法或信号处理时经常会用到的数学直觉。是一道非常经典题目！观察一下，方程的形式是：

## 二、 二阶可降阶微分方程 (Reducible Second-Order DE)

针对无法直接求解的二阶方程，通过变量代换降为一阶。

| 类型       | 形式                      | SOP (解题步骤)                            |
| ---------- | ------------------------- | ----------------------------------------- |
| **类型 ①** | $y'' = f(x)$              | 直接连续积分两次。                        |
| **类型 ②** | $y'' = f(x, y')$ (缺 $y$) | 令 $y' = p$，则 $y'' = \frac{dp}{dx}$。   |
| **类型 ③** | $y'' = f(y, y')$ (缺 $x$) | 令 $y' = p$，则 $y'' = p \frac{dp}{dy}$。 |

**例题（类型 ①）**：$y'' = \sin x$

- $y' = \int \sin x dx = -\cos x + C_1$
- $y = \int (-\cos x + C_1) dx = -\sin x + C_1x + C_2$

**例题（类型 ②）**：$(1+x^2)y'' = 2xy'$

- 设 $y' = p$，则 $(1+x^2)\frac{dp}{dx} = 2xp \implies \frac{dp}{p} = \frac{2x}{1+x^2}dx$
- 积分得 $\ln|p| = \ln(1+x^2) + C \implies p = C_1(1+x^2)$
- 还原 $y' = C_1(1+x^2) \implies y = C_1(x + \frac{1}{3}x^3) + C_2$

**例题（类型 ③）**：$2yy'' + (y')^2 = 0$

- 设 $y'=p, y''=p\frac{dp}{dy}$
- 代入：$2yp\frac{dp}{dy} + p^2 = 0 \implies 2y\frac{dp}{dy} = -p$
- 分离变量积分求解。

> ### 类型三推导步骤
>
> 1. **定义换元：**
>
>    设 $y' = \frac{dy}{dx} = p$。这里我们将 $p$ 视为 $y$ 的函数，即 $p = p(y)$。
>
> 2. **对 $x$ 求导：**
>
>    根据定义，$y'' = \frac{d}{dx}(y') = \frac{dp}{dx}$。
>
> 3. **应用链式法则：**
>
>    由于我们想把 $y''$ 表示为关于 $y$ 的形式，我们将 $\frac{dp}{dx}$ 展开：
>
>    $$y'' = \frac{dp}{dy} \cdot \frac{dy}{dx}$$
>
> 4. **代回定义：**
>
>    因为前面已经设定了 $\frac{dy}{dx} = p$，所以直接代入上式：
>
>    $$y'' = \frac{dp}{dy} \cdot p = p \frac{dp}{dy}$$

# 三、 二阶常系数线性微分方程

## 1. 齐次方程：$y'' + py' + qy = 0$

通过特征方程 $r^2 + pr + q = 0$ 的根 $\Delta = p^2 - 4q$ 来确定解的形式：

| 特征根 $r_1, r_2$ 的情况                | 通解 $y$ 的形式                                       |
| --------------------------------------- | ----------------------------------------------------- |
| 有两个不相等的实根 $r_1 \neq r_2$       | $y = C_1e^{r_1x} + C_2e^{r_2x}$                       |
| 有两个相等的实根 $r_1 = r_2 = r$        | $y = (C_1 + C_2x)e^{rx}$                              |
| 有一对共轭复根 $r = \alpha \pm \beta i$ | $y = e^{\alpha x}(C_1\cos \beta x + C_2\sin \beta x)$ |

## 2. 非齐次方程

方程形式为：

$$(2) \quad y'' + ay' + by = f(x) \quad (f(x) \neq 0, \text{ $a, b$ 为常数})$$

其通解构造为：**齐次方程通解 + 一个特解 (**$Y_0$**)**。

### 解法 (I)：未定系数法 (Method of Undetermined Coefficients)

当非同齐次项 $f(x)$ 具有特定形式时，可以通过“猜测”特解的形式来求解：

| $f(x)$ 的形式                                                | 猜测的特解形式                                  |
| ------------------------------------------------------------ | ----------------------------------------------- |
| (i) $a + be^{\alpha x}$                                      | $A + Be^{\alpha x}$                             |
| (ii) $a \cos \alpha x + b \sin \alpha x$                     | $A \cos \alpha x + B \sin \alpha x$             |
| (iii) $ae^{\alpha x} \sin \beta x$ 或 $ae^{\alpha x} \cos \beta x$ | $e^{\alpha x}(A \cos \beta x + B \sin \beta x)$ |
| (iv) 多项式                                                  | 相同次数的多项式                                |

*注：如果猜测的形式与齐次解冲突，则需要根据重根情况乘上* $x$ *或* $x^2$*。*

### 解法 (II)：常数变易法 (Variation of Parameters)

当 $f(x)$ **形式复杂无法猜测时**，设齐次方程的两个独立解为 $y_1, y_2$，特解为 $Y_0 = u_1 y_1 + u_2 y_2$。 公式如下：

$$u_1(x) = \int \frac{-y_2 f(x)}{W(y_1, y_2)} dx, \quad u_2(x) = \int \frac{y_1 f(x)}{W(y_1, y_2)} dx$$

其中分母 $W(y_1, y_2) = y_1 y_2' - y_1' y_2$ 被称为 **朗斯基行列式 (Wronskian)**。



## 2. 变系数线性微分方程

方程形式为：

$$(3) \quad L(y) = y'' + p(x)y' + q(x)y = f(x)$$

### 解法 (III)：降阶法 (Reduction of Order)

**已知齐次方程** $L(y)=0$ **的一个非零特解** $v$ **时：**

1. 令 $y = uv$。

2. 代入方程整理后可得到关于 $u$ 的二阶导数方程。

3. 设 $w = \frac{du}{dx}$，方程转化为一阶线性微分方程：

   $$\frac{dw}{dx} + \left( \frac{2}{v} v' + p \right) w = \frac{f}{v}$$

   这样通过求出 $w$ 进而求出 $u$ 和 $y$。

### 解法 (IV)：广义常数变易法

**已知齐次方程** $L(y)=0$ **的两个独立特解** $y_1, y_2$ **时：** 这种情况下，即使系数 $p(x), q(x)$ 不是常数，依然可以直接套用解法 (II) 中的公式来求 $u_1, u_2$，从而构造通解：

$$y = C_1 y_1 + C_2 y_2 + u_1 y_1 + u_2 y_2$$

## 核心概念总结

- **齐次 vs 非齐次**：非齐次方程的解总是由对应的齐次方程解加上一个特解组成。
- **朗斯基行列式**：它是判断解是否线性相关的关键，也是常数变易法公式的分母。
- **特解的寻找**：待定系数法最快（但适用范围窄），常数变易法最万能（但积分可能很难）。

## 3. 典型例题解析

### 例题 A：常数变易法（解法 II）

**题目：** $y'' + y = \sec x$

1. **齐次解**：$y_h = C_1 \cos x + C_2 \sin x$。
2. **行列式**：$W = \cos^2 x + \sin^2 x = 1$。
3. **特解系数**：$u_1 = -\int \tan x dx = \ln|\cos x|$；$u_2 = \int 1 dx = x$。
4. **通解**：$y = C_1 \cos x + C_2 \sin x + \cos x \ln|\cos x| + x \sin x$。

### 例题 B：降阶法（解法 III）

**题目：** $x^2 y'' - xy' + y = x^2$，已知齐次解 $y=x$。

1. **变量代换**：设 $y = ux$，则 $y' = u'x + u$，$y'' = u''x + 2u'$。
2. **代入化简**： $x^2(u''x + 2u') - x(u'x + u) + ux = x^2$ 化简得：$x^3 u'' + x^2 u' = x^2 \Rightarrow xu'' + u' = 1$。
3. **降阶**：设 $w = u'$，得一阶方程 $w' + \frac{1}{x}w = \frac{1}{x}$。
4. **解一阶方程**：积分因子为 $x$，解得 $w = 1 + \frac{C_1}{x}$。
5. **积分还原**： $u = \int (1 + \frac{C_1}{x}) dx = x + C_1 \ln |x| + C_2$。
6. **最终通解**： $y = ux = x^2 + C_1 x \ln |x| + C_2 x$。

## 核心概念总结

- **已知解的重要性**：在处理变系数方程时，如果能通过观察或题目给出一个齐次特解，解题难度会大幅降低。
- **降阶法的本质**：通过 $y=uv$ 的代换，消去了方程中的 $u$ 项，使得二阶方程变成了关于 $u'$ 的一阶方程。
