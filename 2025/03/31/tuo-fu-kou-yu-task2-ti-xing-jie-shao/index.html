一、Task2🌟

  • Task 2解锁前提：掌握Task 1
  • 建议学习方式：巩固Task 1同时学习Task 2
  • 高效学习方法：重点关注并优先备考重点题型
  • 学习新阶段注意事项：避免同时学习多个题型导致负担过重
  • Task 2内容预告：接下来将讲解Task 2的相关内容
1.Task Description
1) Task 2介绍及如何解题

  • 视频内容概述：第二题介绍及解题方法
  • 第二题介绍：了解第二题的题型与要求
  • 解题准备：对第二题有基本认识
2) 如何回答Task 2

  • 视频内容概述：第二个视频将讲解Task 2回答方法
  • 答题结构重要性：知道答题结构并不等于完全掌握
  • 答题中可能遇到的问题：语言不准确、不流畅，信息不全面，逻辑不清晰等
  • 问题存在原因：个人能力不足
3) Task 2所需技能

  • 视频目的：分析Task 2考察的核心能力
  • 自检自测：评估自身能力水平
  • 后续步骤：确定并提升不足能力
4) 提升Task 2所需技能

  • 提升Task 2能力的重要性：涉及多种能力，视频系列较长
  • 视频学习建议：针对不足能力仔细观看并完成作业，已掌握能力可跳过
  • 网络学习优势：可选择性学习，节省时间，提高效率
  • 能力与题型结合：能力提升后，结合题型练习，达到流利回答目标
5) Review与行动计划

  • 复习与检测：通过新题测试是否达标
  • 后续备考：提供第二题解决方案
6) Task 2具体题型描述：Fit & Explain

  • Task 2题型介绍开始：开始介绍第二题题型
  • Task 2题型展示方式：通过表格展示
7) Task 2的读、听、说要求

  • Task 2要求概述：读、听、说三项任务
  • 读的任务：阅读校园通知或学生信件，内容关于校园相关事情，长度80-110词，限时45-50秒
  • 听的任务：听学生对话，讨论所读内容，发表看法
  • 说的任务：总结说话者观点，结合阅读信息
8) Task 2与Task 1的不同点

  • Task 2与Task 1差别一：表达内容不同
    • Task 1：表达自己的观点看法
    • Task 2：描述读到的和听到的内容
  • Task 2与Task 1差别二：考察能力变化
    • 考察能力有变化和不同
  • Task 2与Task 1差别三：时间分配不同
    • Task 1：准备15秒，回答45秒
    • Task 2：准备30秒，回答60秒
2.Required Skills

1) 总结阅读和听力重点的先决条件🌟🌟

  • 总结重点信息的先决条件：读懂阅读和听懂听力
  • 读懂阅读和听懂听力的重要性：是做好题目和获得理想成绩的基础
  • 能力提升途径：系列视频中有专门提升能力的环节
2) 识别重点信息的能力

  • 识别重点信息能力的定义：从众多细节中识别出重要信息
  • 真正的读懂：不仅明白字面意思，还能识别出重点
  • 总结内容的方法：在读和听的同时记笔记
3) 记笔记的重要性

  • 记录重点的必要性：作为答题依据
  • 总结的作用：明确考察内容
  • 对比发现：考察内容与第一题不同
4) 第二题与第一题考察能力的区别

  • 第一题考察能力：给出并展开原因
  • 第二题考察能力：与第一题截然不同
  • 备考策略：分清楚备考先后顺序和主次，先备考哪一个都可以
5) 语言能力的考察

  • 语言能力考察概述：是英语考试的要求
  • 考察内容：词汇使用、语法准确性
  • 语言表现：流畅性、句子流利度
3.应用案例

1) 例题
  • 例题介绍：阅读文章背景
  • 阅读任务：总结建议及实施方式
  • 阅读时间：45秒或50秒
  • 阅读后任务：一分半钟找答案
  • 生词讲解：regularly、modernize、stimulate
  • 阅读建议内容：改善宿舍厨房使用
  • 建议实施方式：更新炊具、提供免费烹饪课
  • 听力任务：判断学生观点及原因
  • 听力内容：男生同意建议，分享个人经历
  • 同意原因一：厨房设施待提升
  • 同意原因二：学生需学习烹饪技能
  • 题目要求：总结提议，解释同意或反对原因
4.官方指南🌟🌟🌟

  • 题目要求：结合并总结信息
  • 阅读总结内容：proposal描述及两个细节
  • 听力总结内容：说话者观点及支持或反对阅读的信息
  • 答题要点：包含proposal、原因细节、说话者观点及原因展开
5.高分解答示例

1) 解答包含的内容点

  • 回答主题：学生建议学校改善宿舍厨房使用
  • 建议内容：现代化厨具、提供免费烹饪课
  • 男生支持原因：曾有不良体验（想热茶无杯子）、新厨具会带来改变
  • 烹饪课受欢迎原因：学生不会烹饪、需要技能、有人愿意教、有趣
2) 高分回答的标准

  • 高分回答的信息量：包含足够的信息，而非全部信息
  • 高分回答的语言要求：词汇准确、流畅
  • 高分回答的表达：清晰易懂，容易让考官理解
6.作业

  • 作业内容一：听高分范例回答
  • 作业要求一：思考包含的信息及未包含信息的原因
  • 作业内容二：分析回答清晰易懂的原因
  • 作业要求二：总结出三条原因
