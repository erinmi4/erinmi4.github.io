# 欢迎来到我的博客

这是我使用 Hexo 框架和主题模板创建的第一篇博客文章。在这里，我会记录技术学习、项目经验，以及一些阶段性的想法。

## Hexo 简介

[Hexo](https://hexo.io/) 是一个快速、简洁且高效的博客框架。它支持用 Markdown 编写内容，也很适合部署到 GitHub Pages 这类静态托管平台。

### Hexo 的优点

1. 简单高效，只需要少量命令就能快速启动博客。
2. 原生支持 Markdown，写作门槛低。
3. 插件生态丰富，扩展空间大。
4. 主题选择很多，适合快速搭建个人站点。
