### 立命馆大学情报理工学募集要项解读
原始募集要项：
https://www.ritsumei.ac.jp/gr/exam/point/202704_ag_science_engineering.pdf

这份 PDF 覆盖的不只是你最关心的那一条线，它同时包含：

- `博士课程前期课程（日本语基准）`
- `博士课程后期课程（日本语基准）`
- 对应的 `一般 / 社会人 / 留学生 / 学内 / 飞び級 / 转入学`

这篇文章只解读：

- `博士课程前期课程`
- `一般入学试验（日本语基准）`

也就是通常修考时最常看的那条线。

先把最容易看混的一点说清楚：立命馆这份要项里，`一般入学试验` 实际上有 `2026 年 8 月实施` 和 `2027 年 2 月实施` 两轮。它们都属于一般入试，不是一个是一般、一个是补录。

### 先看最重要的结论

- 研究科是 `情报理工学研究科`
- 前期课程下有两个课程方向：`计算机科学コース` 和 `人间情报科学コース`
- `一般入学试验（日本语基准）` 的募集人数写的是 `10 名`
- 一般入试有两轮：`2026-08-25` 和 `2027-02-09`
- Web 出愿通过 `Ritsu-Mate` 进行，而且只会在出愿首日 `10:00（日本时间）` 开放
- 立命馆这条线最特别的硬门槛是：`出愿前必须先联系希望指导教员，并先交研究计划书`
- 你最后交给事务室的研究计划书，不是你自己单独写完就行，而是 `要拿到教员签字或盖章后的版本`
- 一般入试的出愿材料 `只接受邮送`
- 邮寄标准是 `出愿期间最终日消印有效`，但必须走 `简易书留速达`，而且 `不能投邮筒`
- 英语不考校内笔试，而是直接把 `TOEIC L&R` 换算成 `150 分`
- 一般入试可以用 `TOEIC L&R 公开测试`，也可以用 `TOEIC L&R IP 测试`，但 `在线 IP 测试不可`
- 如果英语成绩最后没交上，`英语直接按 0 分`

如果只记一件事，那就是：这份募集要项里最容易踩坑的，不是笔试科目本身，而是 `事前联系老师 + 拿回已签字研究计划书` 这一步。

### 这条一般入试到底招什么

立命馆在表格里写的是：

- 专攻：`情报理工学`
- 课程：`计算机科学`、`人间情报科学`
- 方式：`一般入学试验（日本语基准）`
- 募集人数：`10 名`

这里有两个值得单独提醒的点：

第一，这个 `10 名` 是写在一般入试这一栏里的数字，不要和：

- `外国人留学生入学试验（日本语基准）10 名`
- `学内进学入学试验 120 名`

混在一起看。

第二，原文没有把这 `10 名` 再拆成“8 月轮多少、2 月轮多少”，所以更稳妥的理解是：

- 一般入试这一方式总共写 `10 名`
- 但具体每轮录多少，学校并没有在这份 PDF 里继续细分

### 最关键的时间线

一般入试最值得盯住的是两轮时间。

| 时间 | 事项 | 备注 |
| --- | --- | --- |
| `2026-06-05` | 个别审查申出期限 | 仅适用于 `8 月一般入试` 的特定资格者 |
| `2026-06-25 10:00` 至 `2026-07-09` | 8 月一般入试出愿期 | `Ritsu-Mate` 首日 10:00 才开放 |
| `2026-08-03` | 8 月一般入试英语成绩补交截止 | `消印有效` |
| `2026-08-25` | 8 月一般入试考试日 | 笔试 + 面试 |
| `2026-08-28` | 8 月一般入试预备日 | 仅灾害等特殊情况延期时使用 |
| `2026-09-10 10:00` | 8 月一般入试放榜 | 在 `Ritsu-Mate` 查看 |
| `2026-11-27` | 个别审查申出期限 | 仅适用于 `2 月一般入试` 的特定资格者 |
| `2027-01-07 10:00` 至 `2027-01-21` | 2 月一般入试出愿期 | 同样从首日 10:00 开始 |
| `2027-02-01` | 2 月一般入试英语成绩补交截止 | `消印有效` |
| `2027-02-09` | 2 月一般入试考试日 | 笔试 + 面试 |
| `2027-02-12` | 2 月一般入试预备日 | 仅特殊情况延期时使用 |
| `2027-02-25 10:00` | 2 月一般入试放榜 | 在 `Ritsu-Mate` 查看 |

这条时间线里最容易被低估的，不是出愿期本身，而是它前面的两步：

1. `先联系老师`
2. `先拿回签字或盖章后的研究计划书`

因为立命馆把这一步写成了正式流程，不是“建议你提前套磁”。

### 立命馆这里为什么一定要先联系老师

这份募集要项最有立命馆特色的地方就在这里。

原文写的是，一般入试出愿者在正式出愿前：

- `必须` 通过邮件等方式联系入学后希望指导的教员
- 就研究内容先做 `事前咨询`
- 同时把 `研究计划书【样式 ISE 1】` 用 `日语` 写好后提交给对方

之后不是你自己继续出愿就行，而是还要：

- 从希望指导教员那里收回 `已经署名或盖章` 的研究计划书
- 再把这份返还后的研究计划书，作为正式出愿材料的一部分交给事务室

募集要项还写得很狠：

- 如果这份研究计划书在出愿期间内 `不能作为出愿书类提交到事务室`
- 那就 `不认可出愿`

也就是说，这里不是“先联系老师会更稳”，而是：

- `先联系老师`
- `先交研究计划书`
- `先拿回已签字版本`

这三步，本身就是一般入试流程的一部分。

### 如果不知道怎么联系老师

立命馆在 PDF 里其实把联络方式写得很具体：

- 先去研究科官网或研究者数据库找公开邮箱
- 没公开邮箱的话，就发邮件给事务室 `gsise@st.ritsumei.ac.jp`

它还给了联络时建议写的内容：

- 姓名
- 希望入学时间和入试方式
- 希望课程序列
- 言语基准
- 希望指导教员名

建议附的材料是：

- `研究计划书【样式 ISE 1】`
- `履历书（自由格式）`

所以这一步不是靠猜，学校已经把最小可执行流程给出来了。

### 正式出愿怎么走

一般入试的正式出愿分成两块：

#### 1. 先在 `Ritsu-Mate` 做 Web 出愿和检定料支付

两轮时间分别是：

- 8 月实施：`2026-06-25 10:00` 到 `2026-07-09`
- 2 月实施：`2027-01-07 10:00` 到 `2027-01-21`

检定料是：

- `35,000 日元`

可用的支付方式包括：

- 信用卡
- 便利店
- PayPal
- Alipay
- 海外汇款

#### 2. 再把纸质材料邮寄出去

一般入试这里写得很明确：

- `只接受邮送`
- `出愿期间最终日消印有效`
- 日本国内寄送要用 `简易书留速达邮便`
- 必须从 `邮便局窗口` 寄出
- `不可投递到邮筒`

另外还要记得：

- 用 A4 不折叠的信封
- 从 `Ritsu-Mate` 下载并打印 `宛名ラベル`
- 把所有出愿材料一次性装进去

和神户大学那种 `必着` 型学校不一样，立命馆这里是 `消印有效`。这是两种完全不同的截止逻辑，不要套错。

### 一般入试要交哪些材料

对一般入试来说，核心材料是这些：

- `入学试验志愿票`
- `毕业（预计毕业）证明书`
- `成绩证明书`
- `研究计划书【样式 ISE 1】`
- `英语能力试验成绩证明书`

按募集要项，研究计划书还有两个硬条件：

- 必须是 `日语` 撰写
- 必须带有 `希望指导教员的署名或盖章`

除此之外，下面这些是 `仅适用于特定情况`：

- `外国籍志愿者信息`
- `学位授与证明书`
- `教育职员免许状授与证明书`

### 如果你是中国大学出身，要注意证书规则变了

这份 PDF 也单独写了中国高校相关要求。

如果你是：

- 中国大学已毕业者
- 或中国大学毕业预计者

那 `毕业（预计毕业）证明书` 这块，不是简单交学校开的那张就完了，而是要按募集要项要求改成交：

- `Online Verification Report of Higher Education Qualification Certificate`
- 或 `Online Verification Report of Student Record`

也就是说，这里和很多日本大学一样，对中国学历走的是在线认证路线，不能想当然按普通原件理解。

### 英语要求怎么理解

立命馆这一条线的英语逻辑很清楚，而且和别的学校不太一样。

#### 1. 英语直接计分，不是参考项

一般入试的总评里，英语是单独一块：

- `外国语（英语）150 分`

学校会把 `TOEIC L&R` 换算成 `150 分满分` 后，直接加进笔试成绩里。

所以它不是“提交了更好”，而是实打实参与总分计算。

#### 2. 一般入试可接受哪些 TOEIC

募集要项对一般入试写的是：

- `TOEIC L&R 公开测试`：可用
- `TOEIC L&R IP 测试`：可用
- `TOEIC L&R IP 测试（在线）`：不可用

这一点挺重要，因为有些学校一律不收 IP，而立命馆这里对一般入试是收 `IP` 的，但 `在线 IP` 不收。

#### 3. 有效期怎么卡

学校的原文是：

- 从对应入试 `试验日往前起算 2 年以内`

所以对应到两轮一般入试，可以这样理解：

- 8 月一般入试：按 `2026-08-25` 往前推两年
- 2 月一般入试：按 `2027-02-09` 往前推两年

#### 4. 如果出愿时还没拿到英语成绩

立命馆给了补交窗口，但补交不是无限期的。

你需要先和其他材料一起提交：

- `英语能力试验成绩证明书相关申告书【样式 ISE 3】`

然后再把成绩单邮寄给事务室。一般入试的补交期限是：

- 8 月实施：`2026-08-03`
- 2 月实施：`2027-02-01`

这两个截止标准写的是：

- `消印有效`

#### 5. 如果最后还是没交英语成绩

这份募集要项写得非常直接：

- 一般入试和飞び級入试，英语成绩 `按 0 分处理`

所以不要把“可以先交申告书”理解成“英语这块可以以后再说”。它只是给了你一个短暂补交窗口，不是免交。

### 这条一般入试考什么

立命馆的一般入试选考方式是：

- `书类选考`
- `笔记试验`
- `外国语（英语）`
- `面接试验`

其中最核心的就是笔试和英语。

#### 1. 笔试总分 450 分

笔试分成两组：

数学系科目群：

- `线形代数` 或 `概率・统计`
- 二选一
- 这部分合计 `150 分`

情报系科目群：

- `数据结构与算法`
- `计算机系统`
- `人工知能`
- `数据库`
- `媒体信息处理`

这五门里选三门：

- 每门 `100 分`
- 这部分合计 `300 分`

所以一般入试的笔试组合实际上是：

- 数学组 `2 选 1`
- 情报组 `5 选 3`

#### 2. 英语 150 分

这部分不是现场笔试，而是：

- `TOEIC L&R` 换算分数

#### 3. 面试

面试是：

- 依据研究计划书等出愿材料进行个人面试

也就是说，你前面和老师沟通、以及你交上去的研究计划书，不只是行政材料，也会真正进入后面的面试环节。

### 考试当天怎么排

8 月和 2 月两轮的一般入试，单日内部的时间安排是同一套逻辑：

| 时间 | 内容 |
| --- | --- |
| `9:15-9:30` | 入室与说明 |
| `9:30-11:30` | 笔记试验 |
| `15:00-` | 面接试验 |

试验场是：

- `立命馆大学大阪茨木キャンパス`

不过募集要项同时说明：

- 详细考场安排要到 `Ritsu-Mate` 查看
- 受验票大约在 `试验日 1 周前` 才能下载

所以你不能只看 PDF，就默认自己已经知道具体教室。

### 你是不是还要先做个别审查

不是每个人都需要，但有几类人一定要警觉。

募集要项写到：

- 出愿资格 `⑾` 和 `⑿` 的人
- 以及某些中国学历不满足共通要项基准的人

要在出愿前先进行 `个别审查`。

前期课程一般入试对应的申出期限是：

- 8 月实施：`2026-06-05`
- 2 月实施：`2026-11-27`

如果你属于：

- 非标准 16 年教育路径
- 个别资格认定路径
- 中国 3 年制专科等边界情况

那就不要等到正式出愿时再赌自己能不能报，最好提前按这两个日期去核对。

### 这份募集要项里最容易踩坑的地方

如果把整份 PDF 压缩成几个最关键的坑点，我会觉得是下面这些。

#### 1. “先联系老师”不是建议，是正式流程

立命馆这里不是可做可不做的套磁，而是明确写进流程里的 `事前咨询`。没有这一步，你连完整出愿材料都凑不齐。

#### 2. 研究计划书必须拿回已签字或盖章版本

你交给老师的是第一步，真正用于出愿的是老师返还给你的那一版。这个顺序不能反。

#### 3. 一般入试有两轮

很多人第一次看会只盯着 `8 月`，忽略了同一份要项里还有 `2027 年 2 月` 的一般入试。你如果 8 月来不及，不代表整年没有第二次机会。

#### 4. 一般入试这里是 `消印有效`，不是 `必着`

这个和很多国立大学的处理方式不一样。规划邮寄时，别把别校的习惯直接套过来。

#### 5. TOEIC IP 可以，但在线 IP 不可以

这条非常细，但很容易误判。立命馆不是完全拒绝 IP，而是只拒绝 `在线 IP`。

#### 6. 英语可以补交，但补不上就直接 0 分

它不是“缺材料后补交不影响评分”，而是给你一个短期缓冲窗口。过了窗口，英语就是 `0`。

### 最后怎么用这份募集要项

如果你现在就在准备立命馆大学情报理工学研究科的一般入试，我会建议按这个顺序推进：

1. 先确定自己走 `8 月轮` 还是 `2 月轮`
2. 立刻确认自己是不是需要 `个别审查`
3. 尽快挑老师并发送 `事前咨询` 邮件
4. 先把 `研究计划书` 做出来并拿回签字版
5. 再准备 `Ritsu-Mate` 出愿和纸质邮寄材料
6. 最后把复习重点放到 `数学 2 选 1` 和 `情报 5 选 3`

立命馆这条线的规则本身不算特别绕，但它对流程顺序要求很强。只要你没有在前面的 `教员联系` 和 `研究计划书返还` 这一步翻车，后面的事情就会顺很多。
